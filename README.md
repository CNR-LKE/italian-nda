# Italian NDA NER Dataset

This repository will host the dataset associated with the paper:

> **Comparative evaluation of small language models for named entity recognition in low resource Italian legal texts using non-disclosure agreements**
>
> submitted to *Discover Artificial Intelligence*.

## Overview

The project introduces a manually annotated dataset for **Named Entity Recognition (NER)** in Italian **Non-Disclosure Agreements (NDAs)**.

The dataset was developed within the ICARUS project (*Intelligent Contract Automation for Rethinking User Services*) and focuses on legal-domain entity extraction from Italian contractual documents.

The annotation process was performed with the support of legal-domain experts and includes entity categories such as:

* Parties
* Roles
* Signatories
* Authorized Subjects
* Penalties
* Duration
* Competent Court
* Applicable Law

The dataset is designed to support research on:

* Legal NLP
* Legal Information Extraction
* Named Entity Recognition
* Low-resource legal language processing
* Italian legal AI systems

---

## Dataset Availability

The dataset is not publicly available at the current stage because the associated manuscript is still under peer review.

In the event of acceptance and publication of the paper, the dataset will be released in JSON format through this repository.

The dataset will also be available from the corresponding author upon reasonable request.

---

## Privacy and Anonymization

All contracts included in the dataset were anonymized prior to release in order to comply with privacy and confidentiality requirements.

Sensitive information, including personal names, addresses, company identifiers, and other potentially identifying information, was removed or masked during preprocessing.
